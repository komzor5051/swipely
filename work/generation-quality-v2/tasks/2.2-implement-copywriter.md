# Task 2.2: Implement copywriter agent

## Description

Create the copywriter agent that receives StrategyOutput, original text, ToV guidelines, brief, design preset tone, and content tone. Writes slide content following the strategy structure. Output is raw slide content (titles, content, post_caption) without strict schema enforcement.

## Why

The copywriter focuses solely on writing quality — following the strategy's structure while producing engaging, human-sounding content. Without schema enforcement overhead, the model can focus on creative output. The formatter (Task 2.3) handles structural validation downstream.

## Acceptance Criteria

- [ ] `lib/generation/agents/copywriter.ts` exports `runCopywriter(input: CopywriterInput): Promise<CopywriterOutput>`
- [ ] `lib/generation/prompts/copywriter.ts` exports the copywriter system prompt
- [ ] Prompt injects: design preset tone, content tone (educational/entertaining/provocative/motivational), ToV guidelines, brief
- [ ] Copywriter follows the slideplan from StrategyOutput (respects slide types, angles, element placement)
- [ ] Output includes titles with `<hl>keyword</hl>` tags for highlighting
- [ ] Output includes `postCaption` with hook line + body + CTA structure
- [ ] Uses `callGemini()` with flash-lite model
- [ ] Prompt includes security footer for inter-agent sanitization (D9)
- [ ] Strategist output is sanitized before injection into copywriter prompt (strip instruction-like patterns)

## Verify-smoke

Call copywriter with test StrategyOutput + topic, verify slide content is returned with correct slide count and non-empty titles/content.
