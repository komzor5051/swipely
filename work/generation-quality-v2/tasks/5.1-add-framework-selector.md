# Task 5.1: Add framework selector to generation page

## Description

Add optional framework selector to the generate form UI. Follow existing pattern of tone selector (2-column grid of toggle buttons). Frameworks displayed with Russian labels. Selection sent as `framework` field in API request.

## Why

Users need a way to choose a carousel framework before generation (AC-2). The selector is optional — when nothing is selected, AI picks the best framework automatically (D3).

## Acceptance Criteria

- [ ] Framework selector rendered in the right column of the form, between tone and format selectors
- [ ] 6 frameworks displayed as toggle buttons in 2-column grid with Russian labels
- [ ] Default: none selected (deselectable — clicking active framework deselects it)
- [ ] Selected framework value sent as `framework` field in API request body
- [ ] When no framework selected, `framework` field omitted from request (AI chooses)
- [ ] Framework options imported from `lib/generation/frameworks.ts` (not hardcoded in page)
- [ ] Selector hidden when `preserveText` mode is active (preserve-text skips strategy)
- [ ] Visual style matches existing tone selector buttons

## Verify-user

Open generate page on localhost, verify framework selector appears, can be toggled, and generates carousels with selected framework.
