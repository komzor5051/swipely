# Task 2.1: Implement strategist agent

## Description

Create the strategist agent that receives topic, tone, framework (optional), and template constraints. Returns `StrategyOutput`: hook type with concrete formula, slide-by-slide plan (type, angle, element placement), and key angles. The strategist prompt includes the hook library (15-20 concrete templates) and framework definitions.

## Why

Currently one massive prompt does everything — strategy, writing, and formatting. Separating strategy lets the AI focus on structure and hook selection without the cognitive load of also writing content. This is the first agent in the pipeline (D1).

## Acceptance Criteria

- [ ] `lib/generation/agents/strategist.ts` exports `runStrategist(input: StrategistInput): Promise<StrategyOutput>`
- [ ] `lib/generation/prompts/strategist.ts` exports the strategist system prompt
- [ ] Prompt contains 15-20 concrete hook templates (fill-in-the-blank formulas, not abstract labels like "CONTRARIAN") (D7)
- [ ] When `framework` is provided, strategist constrains structure to that framework
- [ ] When `framework` is null, strategist picks the best framework based on topic/tone
- [ ] `StrategyOutput` includes: hookType, hookFormula, slideplan (array with type/angle/element per slide), keyAngles, ctaType
- [ ] Uses `callGemini()` from `lib/generation/gemini.ts` with flash-lite model
- [ ] Prompt includes security footer for inter-agent sanitization (D9)
- [ ] Output is valid JSON matching `StrategyOutput` type

## Verify-smoke

Call strategist with test topic "5 mistakes in Instagram marketing", verify StrategyOutput JSON structure has all required fields.
