# Plan 01-02: Wire layout into all templates + remove backdrop-filter

<objective>
Connect the AI-assigned layout field from plan 01-01 to all 24 slide templates, and remove all `backdropFilter` / `backdrop-filter` CSS from slide components to fix html-to-image PNG export artifacts.

Purpose: Without this wiring, the layout values Gemini assigns are ignored. Without removing backdrop-filter, exported PNGs will have blur artifacts or invisible overlays.
Output: All templates layout-aware, zero backdrop-filter usage in slide components.
</objective>

This plan completes the circuit:

**backdrop-filter concern:** html-to-image library cannot render `backdrop-filter: blur()` -- it produces blank or miscolored areas in exported PNGs. The decision (from STATE.md) is to replace all backdrop-filter with solid `rgba()` backgrounds that achieve a similar visual effect without the CSS filter.
